# Lyrics-player-buttom
An interactive lyrics player where the user has to push the button to play any music they want. 

## How to Run

You have to make sure to add the music that you want to add by copy pasting the lyrics that you want to add
Make sure to add a comma after you close the parentheses. 

```
"I know it hurts you (yeah)",
```
Then run the program, press the 

```
more pain
```

buttom.
